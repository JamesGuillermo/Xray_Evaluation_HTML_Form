STEP 1: CREATE GOOGLE SHEET (5 minutes)
================================================================================

1. Go to: https://sheets.google.com
2. Click: "+ Blank"
3. Name it: "Xray_Machine_Evaluations"
4. Rename tab: "Responses"
5. Copy Sheet ID from URL (between /d/ and /edit)
6. Click "Share" ‚Üí "Anyone with the link" ‚Üí "Done"

Your Sheet ID looks like: 1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms

================================================================================
STEP 2: DEPLOY APPS SCRIPT (15 minutes)
================================================================================

1. In your Google Sheet: Extensions ‚Üí Apps Script
2. Delete default code
3. Open Code.gs from your project folder
4. Copy ALL the code and paste into Apps Script editor
5. Find line: const SHEET_ID = 'YOUR_GOOGLE_SHEET_ID_HERE';
6. Replace with your actual Sheet ID (keep the quotes)
7. Save: Ctrl+S (name project: "Xray Form Backend")
8. Function dropdown ‚Üí Select "testSheetAccess"
9. Click Run (‚ñ∂Ô∏è)
10. Grant permissions when prompted:
    - Review permissions
    - Choose your account
    - Advanced ‚Üí Go to Xray Form Backend (unsafe)
    - Allow
11. Check logs: View ‚Üí Logs (should show "Successfully accessed sheet")
12. Function dropdown ‚Üí Select "testFormSubmission"
13. Click Run
14. Check your Google Sheet - should have test data row (delete it)
15. Click: Deploy ‚Üí New deployment
16. Click gear icon ‚Üí Select type: Web app
17. Description: "X-ray Form Submission Handler"
18. Execute as: Me
19. Who has access: Anyone
20. Click Deploy (authorize again if prompted)
21. COPY the web app URL (save it - you need this!)

URL format: https://script.google.com/macros/s/AKfycbw.../exec

================================================================================
STEP 3: UPDATE HTML FORM (3 minutes)
================================================================================

1. Open index.html in text editor (VS Code, Notepad++, etc.)
2. Find (around line 610-615):
   const GOOGLE_SCRIPT_URL = 'YOUR_GOOGLE_APPS_SCRIPT_WEB_APP_URL_HERE';
3. Replace with your web app URL from Step 2
4. Keep the quotes!
5. Save file: Ctrl+S

Example:
const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/AKfycbw1234567890/exec';

================================================================================
STEP 4: TEST (5 minutes)
================================================================================

1. Open terminal/command prompt in your project folder
2. Run: python -m http.server 8000
   OR double-click: start_server.bat (Windows)
3. Open browser: http://localhost:8000/index.html
4. Fill out form:
   - Name: "Test User"
   - Affiliation: "Test Hospital"
   - Date: Today
   - Rate all questions (can use all 3s for quick test)
5. Click "Submit Evaluation"
6. Should see: "‚è≥ Saving to Google Sheets..."
7. Then: "‚úÖ Evaluation Saved Successfully!"
8. Go to your Google Sheet
9. Refresh (F5)
10. Should see new row with your test data
11. Verify totals calculate (columns CM, CN, CO, CP)

If using all 3s: Each total should be 84, Grand total 252

‚ùå TROUBLESHOOTING
================================================================================

PROBLEM: "Failed to load CSV file"
SOLUTION: Use web server (http://), not file:// protocol
‚Üí Run: python -m http.server 8000
‚Üí Or use: start_server.bat

PROBLEM: "Error Saving to Google Sheets"
SOLUTION: Check web app URL
‚Üí Must end with /exec
‚Üí No extra spaces
‚Üí Verify Apps Script deployed with "Anyone" access

PROBLEM: #REF! errors in sheet totals
SOLUTION: Delete test rows and resubmit
‚Üí Apps Script will recreate formulas correctly

PROBLEM: Data in wrong columns
SOLUTION: Clear sheet and let Code.gs auto-create headers
‚Üí Delete all rows
‚Üí Submit test form
‚Üí Headers created automatically

For detailed troubleshooting, see: TESTING_AND_TROUBLESHOOTING.txt
