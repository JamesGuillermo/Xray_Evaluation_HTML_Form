SOLUTION 2: Run a Local Web Server (For testing index.html)
================================================================================

OPTION A: Use the Batch File (Windows - Easiest)
------------------------------------------------
1. Double-click: start_server.bat
2. A command window will open
3. Wait for message: "Serving HTTP on :: port 8000..."
4. Open browser and go to: http://localhost:8000/index.html
5. Form should load with questions!
6. Press Ctrl+C in command window to stop server

OPTION B: Use Python Command (If batch file doesn't work)
---------------------------------------------------------
1. Open Command Prompt (Windows Key + R, type cmd, Enter)
2. Navigate to your folder:
   cd "C:\Users\Lifecare IT Admin\Documents\Programming\XRAY_HTML_Form"
3. Run: python -m http.server 8000
4. Open browser: http://localhost:8000/index.html
5. Press Ctrl+C to stop server

OPTION C: Use Node.js (If you have it installed)
------------------------------------------------
1. Open Command Prompt
2. Navigate to your folder
3. Run: npx http-server -p 8000
4. Open browser: http://localhost:8000/index.html

================================================================================
TROUBLESHOOTING
================================================================================

Problem: start_server.bat says "Python is not installed"
Solution: 
   - Download Python from: https://python.org/downloads
   - Install it
   - Run start_server.bat again

Problem: Port 8000 already in use
Solution:
   - Edit start_server.bat
   - Change 8000 to 8080 or another number
   - Or close other programs using that port

Problem: Form still won't load questions
Solution:
   - Use index_offline.html instead
   - It will ask you to upload the CSV file manually
   - Works without any server!

Problem: CSV validator still shows errors
Solution:
   - Make sure you're using the UPDATED csv_validator.html
   - The fix was applied to match special characters
   - Try refreshing the page (Ctrl+F5)
