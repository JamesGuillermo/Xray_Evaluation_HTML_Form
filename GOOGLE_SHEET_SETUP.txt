5. ADDING FORMULAS FOR TOTALS
================================================================================

The Code.gs script automatically adds formulas for each row.
But if you want to understand or manually add them, here's how:

FOR ROW 2 (first data row):
---------------------------

Column CM (BullsEye Total):
=SUMPRODUCT((MOD(COLUMN(E2:CJ2)-COLUMN(E2),3)=0)*(E2:CJ2))

This sums every 3rd column starting from E (BullsEye ratings only)

Column CN (Aimed Total):
=SUMPRODUCT((MOD(COLUMN(F2:CK2)-COLUMN(F2),3)=0)*(F2:CK2))

This sums every 3rd column starting from F (Aimed ratings only)

Column CO (GE Total):
=SUMPRODUCT((MOD(COLUMN(G2:CL2)-COLUMN(G2),3)=0)*(G2:CL2))

This sums every 3rd column starting from G (GE ratings only)

Column CP (Grand Total):
=CM2+CN2+CO2

This sums all three machine totals

EXPLANATION:
The SUMPRODUCT formula uses MOD to select every 3rd column:
- MOD(COLUMN(E2:CJ2)-COLUMN(E2),3)=0 creates a pattern: TRUE, FALSE, FALSE, TRUE, FALSE, FALSE...
- This multiplies by the cell value only when TRUE (every 3rd column)
- Result: Only BullsEye columns are summed

BASIC STATISTICS:
-----------------
Add these below your data (e.g., after row 100):

Average Scores:
Row 101: =AVERAGE(CM2:CM100)  // Average BullsEye total
Row 101: =AVERAGE(CN2:CN100)  // Average Aimed total
Row 101: =AVERAGE(CO2:CO100)  // Average GE total

Highest Score:
=MAX(CM2:CM100)  // Highest BullsEye total

Lowest Score:
=MIN(CM2:CM100)  // Lowest BullsEye total

Response Count:
=COUNTA(B2:B100)  // Number of submissions

CATEGORY ANALYSIS:
------------------
To analyze by category (based on CSV structure):

X-ray Amount (Q1-Q4):

9. TROUBLESHOOTING
================================================================================

ISSUE: Headers don't match expected layout
-------------------------------------------
Solution: Let Code.gs auto-create headers, or copy the tab-separated list from Section 4 exactly.

ISSUE: Formulas showing #REF! error
------------------------------------
Solution: Check that your column references match the actual layout. E2:CJ2 should span all BullsEye columns.

ISSUE: Totals showing incorrect numbers
----------------------------------------
Solution: Verify that ratings are numeric (0-5) and not text. Check for blank cells.

ISSUE: Can't see all columns
-----------------------------
Solution: Scroll right or zoom out (Ctrl + mouse wheel). The sheet is wide (91 columns).

ISSUE: Slow performance with many rows
---------------------------------------
Solution: 
- Limit conditional formatting ranges
- Remove array formulas if not needed
- Consider archiving old data to another sheet

ISSUE: Duplicate submissions
-----------------------------
Solution: Check "Timestamp" column for exact duplicates. Use Data ‚Üí Remove duplicates if needed.

ISSUE: Missing data in some columns
------------------------------------
Solution: Check that all 28 questions are being filled in the HTML form. Review the question counter in index.html.
