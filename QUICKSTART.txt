================================================================================
QUICK START GUIDE - X-RAY REVIEW FORM
================================================================================

This is the FASTEST way to get your form online. Follow these steps in order.

================================================================================
STEP 1: INSTALL PREREQUISITES (5 MINUTES)
================================================================================

A. Install Node.js:
   1. Go to: https://nodejs.org/
   2. Download the LTS version
   3. Run the installer
   4. Click "Next" through all options
   5. Wait for installation to complete

B. Install Firebase CLI:
   1. Open Command Prompt (Windows Key + R, type "cmd", press Enter)
   2. Type: npm install -g firebase-tools
   3. Press Enter and wait (2-3 minutes)
   4. Verify by typing: firebase --version


================================================================================
STEP 2: CREATE FIREBASE PROJECT (3 MINUTES)
================================================================================

1. Go to: https://console.firebase.google.com/
2. Sign in with your Google account
3. Click "Add project" or "Create a project"
4. Project name: xray-review-form (or your choice)
5. Click "Continue"
6. Disable Google Analytics (unless you need it)
7. Click "Create project"
8. Wait for setup to complete
9. Click "Continue"


================================================================================
STEP 3: PREPARE YOUR FILES (2 MINUTES)
================================================================================

1. Create a new folder on your desktop: "xray-form"
2. Copy these files into that folder:
   - index.html
   - Xray_Evaluation_Questions_Simple.csv
   - firebase.json (optional, will be auto-created)


================================================================================
STEP 4: INITIALIZE FIREBASE (3 MINUTES)
================================================================================

1. Open Command Prompt
2. Navigate to your folder:
   cd C:\Users\YourUsername\Desktop\xray-form
   
3. Login to Firebase:
   firebase login
   (Browser will open - sign in and allow access)

4. Initialize Firebase:
   firebase init hosting

5. Answer these questions:
   ? Please select an option: Use an existing project
   ? Select a default Firebase project: [Choose your project]
   ? What do you want to use as your public directory? public
   ? Configure as a single-page app? N
   ? Set up automatic builds and deploys with GitHub? N

6. Firebase created a "public" folder for you


================================================================================
STEP 5: MOVE FILES TO PUBLIC FOLDER (1 MINUTE)
================================================================================

1. Open your xray-form folder
2. Open the "public" subfolder
3. DELETE the index.html that Firebase created
4. COPY your files into the public folder:
   - index.html (your form)
   - Xray_Evaluation_Questions_Simple.csv

Your structure should be:
xray-form/

STEP 6: TEST LOCALLY (1 MINUTE)
================================================================================

1. In Command Prompt, type:
   firebase serve

2. Open your browser
3. Go to: http://localhost:5000
4. Your form should load with questions!
5. Press Ctrl+C in Command Prompt to stop the test


================================================================================
STEP 7: DEPLOY ONLINE (2 MINUTES)
================================================================================

1. In Command Prompt, type:
   firebase deploy

2. Wait for deployment (30-60 seconds)
3. Look for a line like:
   ‚úî Deploy complete!
   Hosting URL: https://xray-review-form.web.app

4. Copy that URL
5. Open it in your browser
6. Your form is now LIVE on the internet!

Share the URL with anyone:

- Works on phones, tablets, computers
- No installation needed
- No login required

COMMON PROBLEMS & QUICK FIXES
================================================================================

‚ùå "firebase: command not found"
   ‚Üí Close and reopen Command Prompt
   ‚Üí Re-run: npm install -g firebase-tools

‚ùå Form shows "Loading..." forever
   ‚Üí Check CSV file is in public folder
   ‚Üí Make sure you're using http://localhost:5000 (not file://)
   ‚Üí Open browser console (F12) for error details

‚ùå Questions don't display
   ‚Üí Re-export CSV from Excel as "CSV (Comma delimited)"
   ‚Üí Make sure CSV has all 6 columns
   ‚Üí Check there are no empty rows

‚ùå Changes don't appear after deploy
   ‚Üí Clear browser cache (Ctrl+Shift+Delete)
   ‚Üí Hard refresh (Ctrl+F5)
   ‚Üí Wait 1-2 minutes for Firebase CDN

‚ùå Can't login to Firebase
   ‚Üí Check your internet connection
   ‚Üí Make sure you're using a Google account
   ‚Üí Try: firebase logout, then firebase login again
